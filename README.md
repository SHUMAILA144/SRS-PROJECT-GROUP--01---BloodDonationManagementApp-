# SRS-Project-Group--01---BloodDonationManagementApp-
Blood Donation Management App.          
Group Members                        Shumaila Zafar                       Rameen Fatima                        Zobia Shaheen 
Project Description                                                     The Blood Donation Management System is web-based application designed to organize the blood donation workflow.It connects donors, blood banks, recipients and hospitals through a centralized platform, ensuring efficient handling of donors data, blood stock, appointments and emergency requests. This system ensures timely blood availability for patients.
