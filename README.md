Blood Donation Management App.        
  
Group Members  <br>                           Shumaila Zafar   <br>                   Rameen Fatima   <br>                    Zobia Shaheen <br>           
Project Description    <br>                                                 The Blood Donation Management System is a web-based application designed to organize the blood donation workflow. It connects donors, blood banks, recipients and hospitals through a centralized platform, ensuring efficient handling of donors data, blood stock, appointments and emergency requests. This system ensures timely blood availability for patients.
